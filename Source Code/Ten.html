CSS BOX MODEL


The CSS Box model defines how elements are rendered and how their dimensions are calculated. It describes the structure of an element as a rectangular box that has content, padding, a border, and a margin. The box model consists of four main components,
which are:

  • Content
  • padding
  • border
  • margin
