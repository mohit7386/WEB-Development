Five Equal Columns with equal width

This is a Bootstrap Grid System in which we can design the columns for different screensizes. It is basically a 12 columns grid system in bootstrap which is build with flexbox you can use these columns if you don't want 12 colums then you use only as per your need and also you can merge the columns with the other columns and make it a bigger column with the help of span 4 ,5 etc.
